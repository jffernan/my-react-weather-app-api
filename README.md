# my-react-weather-app-api

Welcome to My React Weather App, where you can fetch the current weather by entering a location.  To try my app, first clone the React repo here from GitHub at https://github.com/jffernan/my-react-weather-app
Video demo at: https://youtu.be/WA0c3tG6zGo

## Installation

To use this app, first clone from GitHub via 'git clone git@github.com:jffernan/my-react-weather-app.git' at your terminal, 'cd' to the new directory, then run ‘npm install’, and last, run ‘npm start' to launch your http server.  Last, follow the provided http link to open my app in your browser.

Note: Weather data fetched from OpenWeatherMap API. When trying my app, you can use my API Key or sign-up for a FREE API key at https://openweathermap.org/.
